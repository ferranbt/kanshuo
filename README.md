# Kanshuo (看说)

A tool to extract burned-in Chinese subtitles from videos and annotate them for language learning.

## Usage

Extract and annotate subtitles:

```bash
go run internal/process/main.go process --video <url>
```

Run the server:

```bash
go run internal/process/main.go server
```

Start the extension.
